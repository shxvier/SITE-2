Описание проекта:

Функциональное приложение для управления задачами с возможностью добавления, редактирования и удаления задач. Использует Local Storage для сохранения данных между сеансами.

Технологии:
JavaScript ES6+ Local Storage DOM Manipulation
Функциональность:
  • Добавление новых задач с приоритетами
  • Отметка выполненных задач
  • Фильтрация по статусу (все/активные/выполненные)
  • Сохранение данных в Local Storage
  • Адаптивный интерфейс
